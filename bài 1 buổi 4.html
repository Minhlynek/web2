Tìm hiểu HTML


HTML là ngôn ngữ đánh giấu chuẩn cho các trang web


Với HTML, bạn có thể tạo trang web cho riêng mình


HTML rất dễ học - Bạn sẽ thích nó!


Học dễ dàng với HTML "Hãy tự mình thử"


Với trình soạn thảo "Tự thử" của chúng tôi, bạn có thể chỉnh sửa mã HTML và xem kết quả:

Ví dụ

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
</body>
</html>
